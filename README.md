# Input-Output Helper
The Input-Output base library used by NEORV32 Projects and their Ada libraries:
- Wishbone NPU: https://github.com/dipenarathod?tab=repositories
- Wishbone Camera Controller for OV5640: https://github.com/dipenarathod/Wishbone-Camera-Controller-for-OV5640/tree/main

It allows reading/writing registers, packing bytes as a 32-bit word, and other such utilities used by other Wishbone Projects
